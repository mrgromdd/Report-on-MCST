## 7. Скомпилировать приложение Hello World на C++
```shell=   
$ vim hello.cpp
```
Запишем
```cpp=
#include <iostream>

int main(int argc, char **argv)
{
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```
```shell=  
$ g++ -Wall -o hello ~/Gromov_3IB2/hello.cpp
$ ./hello
```
